La Filarmonica di Milano

Challenge

Bridging the gap between a traditional institutional heritage and a digital-native audience. The objective was to engineer a high-engagement interface that translates a rigid orchestral identity into a high-performance, scalable digital product.


Rhythmic Grid & Information Hierarchy

The visual framework is built on a "metronomic" grid system using dashed-line dividers. This structural discipline isn't just aesthetic; it’s a functional tool designed to organize complex schedules and archival data without adding visual weight. I balanced this rigor with rounded UI elements and high-contrast accents to ensure clear visual signifiers and a modern, accessible user flow.


High-Polish Interactive Logic

With a focus on desktop efficiency, I engineered a library of high-fidelity micro-interactions. Every hover state and transition serves as a functional cue, transforming a data-dense concert calendar into a fluid, frictionless experience. This technical precision ensures that the interface feels as responsive and reliable as the live performances it represents.
